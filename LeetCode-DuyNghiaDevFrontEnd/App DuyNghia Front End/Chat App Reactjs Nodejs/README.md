# Realtime Chat App with React, Node.js, Socket.io and MongoDB

- ![Youtube Tutorial](https://tinyurl.com/2jhqn3m4)

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB

## Step by step run project:

- cd server -> nodemon index.js

- cd public -> yarn start

## Trong project này có sử dụng `MongoDB` để làm `Cơ Sở Dữ Liệu`:

- Tên DataBase của project này là `Chat App`

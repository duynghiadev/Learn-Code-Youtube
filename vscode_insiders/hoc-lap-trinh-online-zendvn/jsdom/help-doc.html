How This API Document Is Organized

This API (Application Programming Interface) document has pages corresponding to the items in the navigation bar, described as follows.

Overview

The Overview page is the front page of this API document and provides a list of all HTML tags and corresponding JavaScript classes.

Class/Interface

Each class has its own separate page. Each of these pages has three sections consisting of a class description, fields, methods, and any notes.

Tree (Class Hierarchy)

The hierarchy page contains a list of classes. The classes are organized by inheritance structure .

Index

The Index contains an alphabetic list of all methods and fields.

Prev/Next

These links take you to the next or previous class, interface, package, or related page.

Frames/No Frames

These links show and hide the HTML frames. All pages are available with or without frames.

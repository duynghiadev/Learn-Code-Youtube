# Introduction

The CSS Battle project is a **web application** that allows users to participate in a CSS coding challenge. The backend of the CSS Battle project is built using NestJS, a popular Node.js framework for building scalable and maintainable server-side applications.

# Extends

This repository has been created using the **[Nestjs Starter Repo](/starter-repos/nestjs-starter-repo)** as its foundation.

> **NOTE** Checkout **[Nestjs Starter Repo](/starter-repos/nestjs-starter-repo)** for more information.

## Installation

Clone the CSS Battle project of backend built on NestJS from the Github repository using the following command:

```bash
$ git clone https://github.com/CodieGlot/css-battle.git
```

> **NOTE** **[Github](https://github.com/CodieGlot/css-battle)** code.

## Usage

The CSS Battle project of backend built on NestJS is a web application that allows users to participate in a CSS coding challenge. The backend of the CSS Battle project is built using NestJS and comes with pre-built APIs for handling user authentication, challenge submissions, and leaderboard updates.

Here are some of the key features of the CSS Battle project:

- **User Authentication** - The CSS Battle project comes with pre-built **APIs** for handling user authentication using **JWT tokens**.
- **Challenge Submission** - Users can submit their solutions to the CSS challenges through the APIs provided by the backend.
- **Leaderboard** - The backend updates the **leaderboard** based on the submissions received from the users.

To use the CSS Battle project, simply access the web application from your browser. The frontend of the CSS Battle project is built separately and is not included in this repository. You can build the frontend using any front-end framework or library of your choice.

> **NOTE** Consider exploring **[Netlify](https://www.netlify.com/)** as an option for deploying your project.

## Deployment

To deploy the Css Battle with backend in NestJS, you can use any **cloud hosting service** that supports Node.js applications and Postgres databases. You can build the backend for production using the following command:

```bash
$ npm run build
```

> **NOTE** My **[deployment](https://css-battle-git-main-codieglot.vercel.app/)**.

This will create a production-ready version of the application in the dist directory. You can then upload the contents of the dist directory to your cloud hosting service.

The CSS Battle project of backend built on NestJS is a great way to challenge and improve your CSS coding skills. With its pre-built APIs and leaderboard functionality, you can create a fun and engaging environment for users to participate in CSS challenges.

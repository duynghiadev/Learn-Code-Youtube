## Overview

The **ReactJS Starter Repo** is a boilerplate repository created to help developers quickly scaffold a new ReactJS project. The Starter Repo comes pre-configured with a variety of tools and libraries to help you get started with building your application.

## Getting Started

To use the ReactJS Starter Repository, you need to have Node.js and npm installed on your machine. You can download and install them from [the official website](https://nodejs.org/) â†—.

Once you have installed Node.js and npm, you can clone the repository from GitHub using the following command:

```bash
$ git clone https://github.com/CodieGlot/reactjs-starter-repo.git
```

After cloning the repository, navigate to the project directory and install the dependencies by running the following command:

```bash
$ npm install
```

> **NOTE** The original repository can be accessed/viewed **[here](https://github.com/CodieGlot/reactjs-starter-repo)**.

## Running the app

```bash
# development
$ npm run dev
```

```bash
# production mode
$ npm run build
```

> **NOTE** You can find more information in the **package.json** file

## Deployment

The **ReactJS Starter Repo** is meant to be a starting point for your application, and it does not come with any deployment configurations out of the box. You will need to configure your deployment settings based on the specific requirements of your application.

> **NOTE** You can find more information in ReactJs official website.
